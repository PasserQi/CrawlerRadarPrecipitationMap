# 爬取雷达降水图

> 待优化：多线程爬取

## 功能说明

1. 爬取指定区域内雷达降水图
2. 根据图片的范围对下载数据进行配准
3. 根据用户意向保存指定时间的数据，并自动完成数据拼接

【图例】雷达降水图的图例（来源：小米天气图例）
